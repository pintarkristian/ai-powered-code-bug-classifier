# Data Directory

This folder stores raw and processed datasets for the code bug classifier.

Expected processed CSV format:

```csv
code,label
"int add(int a, int b) { return a + b; }",0
"char buf[10]; strcpy(buf, input);",1
```

Label convention:

- `0` = clean / non-buggy
- `1` = potentially buggy / risky

Recommended layout:

```text
data/raw/        # downloaded or manually collected datasets
data/processed/  # train.csv, valid.csv, test.csv generated by preprocessing
```

Large datasets should not be committed to Git. Keep only lightweight samples or `.gitkeep` files in version control.
